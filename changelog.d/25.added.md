Added SQLite as a first-class durable backend (`skeino[sqlite]`): a `sqlite_path` setting wires both the checkpointer (`AsyncSqliteSaver`) and a new `SqliteMetadataStore` to a SQLite file (or `:memory:`), giving a serverless durable option without Postgres. A lazy `redis` checkpointer builder is also registered (install `langgraph-checkpoint-redis` yourself — it isn't a managed extra). `create_app` now fails loudly when a durable checkpointer is paired with the in-memory metadata store (override with `allow_ephemeral_metadata=True`).
